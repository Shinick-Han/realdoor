# it-015 — T28 (part 2 of 2) — masked-skeleton reasoning wired into the model path

## 1. Target

Backlog **T28** part 2: widen the label model's EYES (hand it the it-014 masked page
skeleton as CONTEXT) without widening its HANDS (it still nominates ONE field from the
closed set; deterministic geometry still reads and type-checks every value). Subsumes
**T27** (ca_dlse piece-rate: model abstains on `hourly_rate` from the table structure,
no hand-coded refusal), closes **T25** (the caption egress gate becomes the furniture
test, so a name can no longer be sent), attempts **T21** (unknown real-form captions),
and fixes **T26** (a timeout no longer poisons the memo). Deterministic headline must be
byte-identical (model off) — non-negotiable.

## 2. The printed evidence

`Rate/Hour` on `ca_dlse_paystub_piecerate` heads a **multi-row** table (Productive /
Non-productive / Rest / Overtime / Double-time, each its own rate) — the it-014 skeleton
prints exactly that structure with every figure blanked. One caption in isolation cannot
tell it from a lone rate; the skeleton can. The T21 carriers (`AVERAGE HOURS PER WEEK`,
`RATE OF PAY OR SALARY`) are real captions the frozen tables never saw.

## 3. The license (exactly one): **printed structure**

The model reads only what the page prints *as structure*: the target caption (already
selected by the extractor's own label detection) plus the masked skeleton (captions,
headers, row-labels, footnote markers — every value a typed placeholder). It nominates a
name; nothing else changes. The value is still read by geometry and must still parse as
the field's type, so a wrong nomination yields an abstention, never a wrong figure.

## 4. The rules (additive; the model path only)

- `core/skeleton.py`: `is_furniture_text(text)` (the context-free egress test),
  `page_sendable_labels(words, position_extension)` and `page_context(words, …)`.
- `core/label_llm.py`: a per-page `_PageContext` (skeleton + sendable set + digest); the
  **unified egress gate** `_sendable` replaces the 48-char `assert_no_values` shape test on
  the send decision (T25) — furniture always, position-qualified only under the extension;
  the skeleton rides the user message as context; the memo key gains the context digest so
  a caption is answered per structure, never cross-contaminated document to document; a
  **timeout is no longer memoized** (T26).
- `core/extract.py`: inside the `label_llm.is_enabled()` guard in `extract_document`, set
  the page context before the scan and clear it after. Two flags:
  `REALDOOR_SKELETON_CONTEXT` (on; consulted only when the model is on) and
  `REALDOOR_SKELETON_POSITION` (**off** — the T21 arm, gated on falsification).

## 5. The hazards, named

- **A value leaves the process inside the skeleton** — it-014's predicate, re-run here
  unchanged (0 leaks), because the wiring must move nothing.
- **A value leaves on the CAPTION channel** — the position extension lets a run that
  anchors a value be sent; it-014 warned a free-floating employer/person name anchors an
  address. **Re-falsified** (`loop/falsify/it-015.py`): the extension leaks **139**
  `person_name` values (a value anchoring a value), so it ships **OFF**; furniture-only is
  0 leaks and is the default. T21's unknown captions are therefore **left on the table**.
- **The model is overconfident** (it-011: 4 wrong). The confirm layer is the backstop;
  measured on every model-on corpus, must stay wrong 0.
- **T26 irreproducibility** — fixed; measurements additionally run at
  `REALDOOR_LABEL_TIMEOUT=30`.
- **Deterministic drift** — impossible by construction: the context is built only inside
  `label_llm.is_enabled()`, false on every deterministic run, so G5 holds.

## 6. Firing predicate

`loop/falsify/it-015.py` — read-only, both outbound channels over all seven corpora
(dev-only where sealed). Channel 1 (skeleton) = it-014's predicate re-run. Channel 2
(caption) = no truth value appears as/inside a sendable caption unless that caption is
furniture-by-string. Reports furniture-only (shippable) and position-extended (gated)
verdicts. Written to `loop/falsification/it-015.json`.
