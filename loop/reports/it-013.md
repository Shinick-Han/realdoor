## The licence shipped

**Printed structure.** Three or more candidates for the same field on one band of
type are a menu: the page offers a choice, and until a mark is made it asserts
nothing, so no run on the band anchors the field (it abstains as before). The band
test is `_on_one_band` — baselines within the *smaller of their type sizes*, the
page's own line pitch — which is exactly what separates seattle's staggered
4.56pt/8.04pt menu (one band) from ca_dlse's 10.68pt/7.68pt table rows (not). No
vocabulary of frequency words; the signal is geometry read through whatever mapper
the pass carries. Threshold is **3** (`MIN_OPTION_CELLS`), so `bonita`'s twin
`GROSS EARN'S` headers and every caption-plus-YTD pair stay untouched. **Marks are
out of scope** — a ticked/circled menu still abstains; reading the mark is a
successor, not a guess about ink a word reader cannot see.

**Printed punctuation** (`REALDOOR_GLOSS_COLON`): a `FREE_TEXT_FIELDS` run with a
colon *inside* it is a caption and its own gloss, never a value — the
terminal-colon rule read one notch further, same scope.
